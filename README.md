# Nerea Qing Services - Sitio Web de Servicios Freelance

Sitio web moderno y responsive para desarrollador freelance.  
Incluye sección de servicios, formulario de contacto y navegación adaptable. 

**Demo:** https://nqingservices.netlify.app/

## Tecnologías
- HTML5
- CSS3 (Flexbox, Grid)

## Características

- Navegación responsive (columnas en móviles, horizontal en escritorio).
- Tres tarjetas de servicios, con iconos y diseño limpio.
- Formulario de contacto.
- Diseño responsive (móvil, tablet y escritorio).
- Scroll Snap integrado en varias secciones.
